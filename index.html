VerseMem   JS Web-API

One of the practices of my faith as a Christian is to learn about and memorize the Bible, so I made this program. I mainly made this in Javascript, and I learned a lot about using web APIs in the process. The Bible is a huge document, so it's not necessarily practical to store and process it in its entirity on your own website or database. Instead, there are APIs available for free that allow you to query a database to get specified sections of the text. So making this program also helped me learn a little about data processing; I had to learn how to turn the data I got from the API into something I could use for my program. Since I also have an interest in foreign languages, I implented this program for a couple different foreign translations of the Bible.
